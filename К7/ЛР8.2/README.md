# Мини-проект: Генератор и анализатор операций
**Лабораторная работа №8 (часть 2)**

Этот проект демонстрирует разделение логики программы на два этапа: автоматическую генерацию данных и их последующую обработку.

## Структура проекта

1.  **generate_data.py** — Скрипт для создания синтетических данных. Он генерирует список из 20 случайных чисел (от 1 до 100) и сохраняет их в текстовый файл `data.txt`.
2.  **main.py** — Скрипт для обработки данных. Он считывает числа из файла и находит пары, сумма которых превышает 150.
3.  **data.txt** — Промежуточный файл с данными (создается автоматически).

## Принцип работы

Программа построена по принципу конвейера данных:
`Генерация (Random) -> Сохранение (File IO) -> Чтение -> Анализ -> Вывод результатов`
