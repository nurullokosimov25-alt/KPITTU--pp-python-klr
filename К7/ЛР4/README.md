Задача 1. Создать систему логирования ошибок математических операций: деление, корень, логарифм. Писать ошибки в math.log
Бизнес логика: Пользователь хочет провести математическую операцию: привилегированные квадратный корень логарифм Если операция прошла успешно → вернуть результат. В случае возникновения ошибки : делиние на 0 корень из отрицательного числа логарифм от 0 или отрицательного числа Ошибка при внесении в файлmath.log с описанием.
Задача 2. Модуль backup.py: • копирование файла • создание архива (zip) • логирование операций
Бизнес логика: Модуль backup.pyизготовления трех вещей:
1.	целизирует музыку
o	если файл существует → копируемый
o	если нет → пишем ошибку в регистрации
2.	Создай ZIP-архив
o	добавить файл в архив
o	если ошибка → логируемый
3.	Логирует все операции
o	успех → ИНФО
o	ошибка → ОШИБКА
o	лог пишет вbackup.log
