Задача 1. Подсчитать количество положительных чисел, пока не встретится ноль (while)
Бизнес-логика решения:
Заводим счётчик — он показывает, сколько положительных чисел ввели. Начинаем вводить числа одно за другим. Если число больше нуля, прибавляем +1 к счётчику. Если число ноль, то останавливаемся. В конце показываем, сколько положительных чисел было введено до нуля.
Задача 2. Найти расстояние между повторениями символа (позиции i и j) Например: s = "abcaab" повтор "a" на позициях 0, 3, 4 → расстояния: 3, 1
Использовать словарь-аккумулятор: last_seen[ch] = позиция 
Бизнес-логика решения: Берём словарь last_seen. В нём будем запоминать, где последний раз встречали каждый символ. Идём по строке слева направо.Для каждого символа: Если видим его впервые — просто запоминаем его позицию. Если уже видели, то: считаем расстояние: текущая позиция минус последняя позиция этого символа; записываем это расстояние в результат ;обновляем позицию символа в словаре. В конце получаем список всех расстояний между повторными появлениями символов.
Задача 3. «Обработка данных о студентах» 
Дано: students = ["Ali_2004", "Zuhra_2003", "mansur_2005", "RUSTAM_2004"] Каждая строка содержит имя + год рождения. Что нужно сделать 1. Разделить строку на имя и год (split("_")). 2. Имя привести к виду: первая буква заглавная, остальные маленькие. 3. Год преобразовать в число. 4. Создать список строк формата: "Имя, возраст: ХХ" (возраст считать как 2025 – год рождения) 5. Вывести список.
Бизнес-логика решения: Берём каждый элемент из списка students. Разделяем его на имя и год рождения по символу _. Имя делаем красивым: первая буква заглавная, остальные маленькие. Год превращаем в число и считаем возраст: 2025 - год. Создаём строку вида "Имя, возраст: ХХ" и добавляем в новый список. После того как пройдём весь список, показываем новый список.
